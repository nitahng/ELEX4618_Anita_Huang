\doxysubsection{Detailed Description}
Base class shared by all ELEX 4618 labs. 

Provides the common “game loop” interface and shared resources for labs that use the TM4\+C123G controller + Open\+CV/\+cvui rendering. Child classes implement hardware I/\+O, state updates, and drawing behavior.

\begin{DoxyNote}{Note}
The joystick percentages ({\ttfamily percentage\+\_\+x}, {\ttfamily percentage\+\_\+y}) represent {\bfseries{analog inputs}} normalized to a percentage range. 

{\ttfamily index} and {\ttfamily findex} are {\bfseries{general-\/purpose state values}} (integer/\+float) used to pass any data from update to draw. 
\end{DoxyNote}

\doxysubsection{Detailed Description}
Declaration of the \doxylink{class_c_control}{CControl} class for serial-\/based I/\+O control. 

This class provides an interface for communicating with external hardware using the \doxylink{class_serial}{Serial} class. It supports sending and receiving digital and analog data for ELEX 4618 Lab 3.

Detailed Description

Abstract base class that defines the lab framework.

Child classes must implement:

  • gpio (read/write controller I/O)
  • update (compute next state)
  • draw (render based on current state)

Update the lab/game state (implemented in child class).

Consumes the latest inputs (e.g., joystick percentages) and updates internal state variables used for drawing and interaction.

Parameters
[in,out]percentage_xJoystick X as a percentage (analog input).
[in,out]percentage_yJoystick Y as a percentage (analog input).
[in,out]colour_button_pressedTrue if the colour/change button is pressed.
[in,out]pt1First point used for drawing (previous/anchor point).
[in,out]pt2Second point used for drawing (current point).
[in,out]indexGeneral-purpose integer value to pass from update() to draw().
[in,out]findexGeneral-purpose float value to pass from update() to draw().
Note
index and findex are not limited to analog inputs—use them for any state you want to carry into draw.

Implemented in CPong, and CSketch.

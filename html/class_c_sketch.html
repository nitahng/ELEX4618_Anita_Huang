Detailed Description

Etch-A-Sketch game implementation.

CSketch inherits from CBase4618 and provides concrete implementations of the gpio(), update(), and draw() methods for the Etch-A-Sketch application. It reads joystick/buttons via the microcontroller and renders drawing output onto an OpenCV canvas.

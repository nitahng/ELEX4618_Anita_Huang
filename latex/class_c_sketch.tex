\doxysubsection{Detailed Description}
Etch-\/\+A-\/\+Sketch game implementation. 

\doxylink{class_c_sketch}{CSketch} inherits from \doxylink{class_c_base4618}{CBase4618} and provides concrete implementations of the \doxylink{class_c_sketch_ad7b6934677df1efc7ee4355e124d6416}{gpio()}, \doxylink{class_c_sketch_a3b7e2fa57fd26e4b812494268a5bbfbb}{update()}, and \doxylink{class_c_sketch_a93f276905cba79da1a18df36d193736f}{draw()} methods for the Etch-\/\+A-\/\+Sketch application. It reads joystick/\+buttons via the microcontroller and renders drawing output onto an Open\+CV canvas.

\doxysubsection{Detailed Description}
Etch-\/\+A-\/\+Sketch game implementation. 

\doxylink{class_c_sketch}{CSketch} inherits from \doxylink{class_c_base4618}{CBase4618} and provides concrete implementations of the \doxylink{class_c_sketch_af1d882d2274d33bd3279c62a87dcc24a}{gpio()}, \doxylink{class_c_sketch_a1ab4eb888473f001e81564f41df5106d}{update()}, and \doxylink{class_c_sketch_a1dc121fe3ed903e4201e026d419bbacf}{draw()} methods for the Etch-\/\+A-\/\+Sketch application. It reads joystick/\+buttons via the microcontroller and renders drawing output onto an Open\+CV canvas.
